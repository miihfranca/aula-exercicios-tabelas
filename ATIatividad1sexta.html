FUNÇÕES DAS TAGS

TABELA
Ela é usada para definir tabelas.

TH
Ele define uma célula cabeçalho do grupo de células da tabela.

TR
Ela é usada para definir uma linha.

TD
É usada para definir uma célula.

THEAD
É usado para dar os destaques dos títulos das colunas de uma tabela.

TBODY
É toda estrutura e conteúdo da tabela.

FOOT
Ela define conteúdos de rodapé dentro de uma tabela e juntá-lo ao corpo e ao cabeçalho.


ALINHAR
Conhecido com (align) serve simplesmente para alinhar.

BORDA
Ele adiciona bordas em qualquer elemento (HTML)

LEGENDA
Conhecido como (caption) representa o título de uma tabela.

ROWSPAN
Ocupa o lugar das próximas células vizinhas abaixo de sua linha.


COLSPAN
Faz com que uma célula ocupe o lugar das próximas células vizinhas ao lado de sua linha.

LARGURA
Ela determina a largura do conteúdo.
